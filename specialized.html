Paint Manufacturing Volume Prediction & Classification Dec 2025
Machine Learning Pipeline (Python, Scikit-learn, Pandas)

Developed a supervised Machine Learning pipeline on a real-world anonymized manufacturing dataset (710 records, 11 features) to predict paint filling volume via regression and classify size categories.

  • Preprocessing: End-to-end handling of missing values, duplicate removal, categorical encoding, feature engineering, unit conversion, and MinMaxScaler feature scaling.
  • Models: Linear & Polynomial Regression (volume) | KNN, Decision Tree, SVM (classification).
  • Evaluation: Achieved high predictive accuracy measured via MAE, MSE, RMSE, R², and confusion matrices. Diagnostics guided by exploratory data analyses.
